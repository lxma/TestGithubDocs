# TestGithubDocs
A test repository that checks how to work with Github Docs

You can jump to the standard version [here](https://github.com/lxma/TestGithubDocs/blob/main/docs/index.md).

You can see the pretty version [here](https://lxma.github.io/TestGithubDocs/).

To get the pretty version, you need to activate the pages:
- In your Github repository, go to Settings->Pages.
- As source, choose "deploy from branch".
- And configure to use the correct folder.
